### Vytvořit na lokálním prostředí

```bash
git init --bare <cesta>
```

> [!WARNING]
> ` <cesta>` = vytvoří úložiště do cesty, musí mít na konci cesty `.git`

### Použít do pracovního prostředí

```bash
git clone <cesta>
```

> [!WARNING]
> ` <cesta>` = adresa k úložišti, musí mít na konci cesty `.git`
>
> > [!NOTE]
> > Může být lokální i online.
