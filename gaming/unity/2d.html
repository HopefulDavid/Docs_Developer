    2D

    Výběr Tilemap x Sprite Renderer

    Tip

    Výkonnostně je lepší použít Tilemap, pokud máte velké množství objektů, které se opakují.

    Velikost obrázků

    Velikosti pro 2D hry:

    • 16x16 px

      Malé objekty

      Například: mince, klíče, atd.

    • 32x32 px

      Středně velké objekty

      Například: postavy, stromy, atd.

    • 64x64 px

      Velké objekty

      Například: budovy, atd.

    Note

    Většinou se používají mocniny čísla 2, protože se lépe pracuje s pamětí.

    Nastavit velikosti

    Například obrázek o velikosti 64x64 pixelů.

    Warning

    Pixel Per Unit (PPU) - kolik pixelů je na jednotku v Unity.

    Je důležité nastavit dle velikosti grafiky, aby byla správně zobrazena.
